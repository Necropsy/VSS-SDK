Instalação e Configuração

O VSS-SDK é a união de seis projetos, o VSS-Vision, VSS-Simulator, VSS-SampleStrategy, VSS-Viewer, VSS-GameLogger e VSS-Joystick. Para possibilitar essa únião utilizamos a mecânica de submódulos do github. Essa etapa é necessária ser executada apenas uma vez.


Instalação Completa

Caso não possua todas as dependências instaladas

No Terminal:

./configure.sh

Instalaçao Parcial

Caso possua todas as dependências instaladas

No Terminal:

./init.sh
./update.sh

Atenção

A instalação foi testada somente nas distros Ubuntu 14.04, Ubuntu 16.04, Debian 8.2 e Debian 8.5.

Atualização

Necessário sempre que houver uma atualização no repositório

No Terminal:

./update.sh

Atenção

A atualização reseta e atualiza todos os projetos, fazendo com que quaisquer modificações em qualquer projeto sejam perdidas.
